Phần hàm số lượng giác


Bài 1:

a) $sin(x + 2) = \dfrac{1}{3}$

b) $sin(3x) = 1$

c) $sin(\dfrac{2x}{3} - \dfrac{\pi}{3}) = 0$

d) $sin(2x + \dfrac{\pi}{9}) = -\dfrac{\sqrt{3}}{2}$

Bài làm:

a) $sin(x + 2) = \dfrac{1}{3}$

Vì $\dfrac{1}{3} = sin(-\dfrac{\pi}{3}) \iff ín(x + 2) = sin(\dfrac{\pi}{3})$

$\iff \Bigg\{\displaylines{x + 2 = -\dfrac{\pi}{3} + 2k\pi~\{k \in Z\} \\\ x + 2 = \pi + \dfrac{\pi}{3} + 2k\pi~\{k \in Z\}}$

$\iff \Bigg\{\displaylines{x = -\dfrac{\pi}{3} + 2k\pi - 2~\{k \in Z\} \\\ x = \pi + \dfrac{\pi}{3} + 2k\pi - 2~\{k \in Z\}}$

b) $sin(3x) = 1$

Vì $1 = sin(\dfrac{\pi}{2}) \iff sin(3x = sin(\dfrac{\pi}{2}))$

$\iff 3x = \dfrac{\pi}{2} + 2k\pi~\{k \in Z\}$

$\iff x = \dfrac{\pi}{6} + \dfrac{2k\pi}{3}~\{k \in Z\}$

c) $sin(\dfrac{2x}{3} - \dfrac{\pi}{3}) = 0$

Vì 0 = sin(0) \iff sin(\dfrac{2x}{3} - \dfrac{\pi}{3}) = sin(0)

$\iff \Bigg\{\displaylines{\dfrac{2x}{3} - \dfrac{\pi}{3} = 2k\pi~\{k \in Z\} \\\ \dfrac{2x}{3} - \dfrac{\pi}{3} = \pi + 2k\pi~\{k \in Z\}}$

$\iff \Bigg\{\displaylines{\dfrac{2x}{3} = 2k\pi + \dfrac{\pi}{3}~\{k \in Z\} \\\ \dfrac{2x}{3} = \pi + 2k\pi + \dfrac{\pi}{3}~\{k \in Z\}}$

$\iff \Bigg\{\displaylines{x = 3k\pi + \dfrac{\pi}{2}~\{k \in Z\} \\\ x = \dfrac{2\pi}{3} + 3k\pi + \dfrac{\pi}{2}~\{k \in Z\}}$

d) $sin(2x + \dfrac{\pi}{9}) = -\dfrac{\sqrt{3}}{2}$

Vì $-\dfrac{\sqrt{3}}{2} = sin(-\dfrac{\pi}{3}) \iff sin(2x + \dfrac{\pi}{9}) = sin(-\dfrac{\pi}{3})$

$\iff \Bigg\{\displaylines{2x + \dfrac{\pi}{9} = -\dfrac{\pi}{3} + 2k\pi~\{k \in Z\} \\\ 2x + \dfrac{\pi}{9} = \pi + \dfrac{\pi}{3} + 2k\pi~\{k \in Z\}}$

$\iff \Bigg\{\displaylines{2x = -\dfrac{4\pi}{9} + 2k\pi~\{k \in Z\} \\\ 2x = \pi + \dfrac{2\pi}{9} + 2k\pi~\{k \in Z\}}$
