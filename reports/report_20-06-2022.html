Phần hàm số lượng giác

Bài 1: giải các phương trình sau:

a) $sin(3x) = -\dfrac{sqrt{3}}{2}$
b) $sin(2x - \dfrac{\pi}{12}) = \dfrac{\sqrt{2}}{2}$

c) $sin(\dfrac{x}{2} + \dfrac{\pi}{18}) = -\dfrac{1}{2}

d) $sin(4x) = \dfrac{2}{3}$

Bài làm:
a)

$\iff 3x = \dfrac{\pi}{6} \iff x = 20$

b)
$\iff 2x - \dfrac{\pi}{12} = \dfrac{\pi}{4} \iff 2x = \dfrac{\pi}}{3} \iff x = \dfrac{\pi}{6}$

c)

$\iff \dfrac{x}{2} + \dfrac{\pi}{18} = -\dfrac{\pi}{6} \iff \dfrac{x}{2} = -\dfrac{2\pi}{9}$

$iff x = \dfrac{4\pi}{2}$

d)

$\iff 4x = 0.73\pi \iff x = 0.18\pi$
