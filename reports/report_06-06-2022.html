Bài 1: Xác định độ dài các trục, tiêu cự, tọa độ các tiêu điểm và các đỉnh của các elip sau:

a) $\dfrac{x^2}{100} + \dfrac{y^2}{36} = 1$

b) $\dfrac{x^2}{169} + \dfrac{y^2}{25} = 1$

Bài làm:

a)

Độ dài trục ngang là : $A_1A_2 = 2a~\iff~A_1A_2 = 2\sqrt{100}~\iff~A_1A_2 = 20$

Độ dài trục ngang là : $B_1B_2 = 2a~\iff~B_1B_2 = 2\sqrt{36}~\iff~B_1B_2 = 12$

Ta có $b^2 = a^2 - c^2$

$\Rightarrow~c = \sqrt{a^2 - b^2}$

$\iff~c = \sqrt{100 - 60}$

$\iff~c = 8$

$\Rightarrow$ tiêu điểm $F_1 = (-8; 0)$

$\Rightarrow$ tiêu điểm $F_2 = (8; 0)$

$\Rightarrow$ tiêu cự $F_1F_2 = 16$

các đỉnh : $A_1 = (-100; 0); A_2 = (100; 0); B_1 = (0; -36); B_2 = (0; 36);$

b)

Độ dài trục ngang là : $A_1A_2 = 2a~\iff~A_1A_2 = 2\sqrt{169}~\iff~A_1A_2 = 26$

Độ dài trục ngang là : $B_1B_2 = 2a~\iff~B_1B_2 = 2\sqrt{25}~\iff~B_1B_2 = 10$

Ta có $b^2 = a^2 - c^2$

$\Rightarrow~c = \sqrt{a^2 - b^2}$

$\iff~c = \sqrt{169 - 25}$

$\iff~c = 12$

$\Rightarrow$ tiêu điểm $F_1 = (-12; 0)$

$\Rightarrow$ tiêu điểm $F_2 = (12; 0)$

$\Rightarrow$ tiêu cự $F_1F_2 = 24$

các đỉnh : $A_1 = (-169; 0); A_2 = (; 0); B_1 = (0; -5); B_2 = (0; 5);$

Bài 2: Lập phương trình đường elip biêt một tiêu điểm $F(2; 0)$ và độ dài trục lớn bằng 10

bài làm:

Độ dài trục lớn $A_1A_2 = 10~\Rightarrow~a = 5$

Tiêu cự $F(2; 0)~\Rightarrow~c = 2$

$\Rightarrow~b^2 = a^2 - c^2~\iff~b = \sqrt{a^2 - c^2}$

$\iff b = \sqrt{5^2 - 2^2}$

$\iff b = \sqrt{21}$

$\Rightarrow$ phương trình cần tìm $\dfrac{x^2}{25} + \dfrac{y^2}{21}$
