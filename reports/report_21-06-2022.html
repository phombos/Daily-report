Bài 1:

a) $cos(x + 3) = \dfrac{1}{3}$

b) $cos(3x - \dfrac{\pi}{4}) = \dfrac{\sqrt{2}}{2}$

c) $cos(2x + \dfrac{\pi}{3}) = \dfrac{\sqrt{3}}{2}$

d) $(2 + cos(x))(3cos(2x) - 1) = 0$

Bài 2:
a) $tan(2x + \dfrac{\pi}{4}) = -1$

b) $tan(\dfrac{x}{2} - \dfrac{\pi}{4}) = tan(\dfrac{\pi}{8})$

Bài làm:

a) Do $cos(x + 3) = \dfrac{1}{3}$ nên $x + 3 = \dfrac{\pi}{2.57}$

$\iff x = \dfrac{\pi}{2.57} - 3$

b) Do $cos(3x - \dfrac{\pi}{4}) = \dfrac{\sqrt{2}}{2}$ nên $3x - \dfrac{\pi}{4} = \dfrac{\pi}{4}$

$\iff 3x = \dfrac{\pi}{2}$

$\iff x = \dfrac{\pi}{6}$

c) Do $cos(2x + \dfrac{\pi}{3}) = \dfrac{\sqrt{3}}{2}$ nên $2x + \dfrac{\pi}{3} = \dfrac{\pi}{6}$

$\iff 2x = \dfrac{5\pi}{12}$

$\iff x = \dfrac{5\pi}{24}$

Bài 2:

a) ĐKXĐ: $cos(2x + \dfrac{\pi}{4}) \neq 0$

$\iff 2x + \dfrac{\pi}{4} \neq k\pi + \dfrac{\pi}{2}~\{k \in Z\}$

$\iff 2x \neq k\pi + \dfrac{\pi}{4}~\{k \in Z\}$

$\iff x \neq \dfrac{k\pi}{2} + \dfrac{\pi}{8}~\{k \in Z\}$

Do $tan(2x + \dfrac{\pi}{4}) = -1$ nên $2x + \dfrac{\pi}{4} = \dfrac{\sqrt{\pi}}{4}$

$\iff 2x = -\dfrac{\pi}{2}$

$\iff x = -\dfrac{\pi}{4}$
