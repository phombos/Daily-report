a) $y = tan(x - \dfrac{\pi}{6})$

b) $y = \dfrac{tan(2x)}{sin(x) + 1} + cot(3x + \dfrac{\pi}{6})$

c) $y = \dfrac{tan(5x)}{sin(4x) - cos(3x)}$

d) $y = cos(x + \dfrac{\pi}{2}) + 1 - \dfrac{cos(4x) + tan(x - \pi)}{sin(x) - 1}$

bài làm:

a)

ĐKXĐ: $cos(x - \dfrac{\pi}{6}) \neq 0$
$x - \dfrac{\pi}{6} \neq k\pi + \dfrac{\pi}{2}~(k \in \Bbb{Z})$

$\iff x \neq k\pi + \dfrac{2\pi}{3}$

$\iff D = \Bbb{R} \backslash~\{k\pi + \dfrac{2\pi}{3}\}$

$ĐKXĐ \Bigg\{\displaylines{cos(2x) \neq 0 \\\ sin(x) + 1 \neq 0 \\\ sin(3x + \dfrac{\pi}{6}) \neq 0}$

$cos(2x) \neq 0 \iff 2x \neq k\pi + \dfrac{\pi}{2}~(k \in \Bbb{Z})$

$\iff x \neq \dfrac{k\pi}{2} + \dfrac{\pi}{4}~(k \in \Bbb{Z})$

$sin(x) + 1 \neq 0 \iff sin(x) \neq -1 \iff x \neq k\pi + \dfrac{pi}{2}~(k \in \Bbb{Z})$

$sin(3x + \dfrac{\pi}{6}) \neq 0 \iff 3x + \dfrac{\pi}{6} \neq k\pi~(k \in \Bbb{Z})$

$\iff 3x \neq k\pi - \dfrac{\pi}{6}$

$\iff x \neq \dfrac{k\pi}{3} - \dfrac{\pi}{18}$
